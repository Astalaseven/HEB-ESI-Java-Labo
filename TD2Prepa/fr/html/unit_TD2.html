    1.2. TD2 Exercices préparatoires

    Objectifs

    Voyons ce que vous avez retenu du TD1.


    La connexion

    Remplissez les blancs.
    Nous allons faire les laboratoires sur une machine qui se nomme linux1,x et qui tourne le système d'exploitation Linux,x Pour s'y connecter, nous utilisons le programme Windows putty,x Nous communiquerons avec la machine en mode console,x grâce à un shell,x qui se nomme bash,x

    Les commandes de base

    La semaine passée, vous avez vu quelques commandes de base. Vous en rappelez-vous ?
    • La commande pour voir le contenu d'un dossier (la liste de ce qu'il contient) est ls,x
    • La commande pour éditer le contenu d'un fichier est vi,x
    • La commande pour changer son mot de passe est passwd,x
    • La commande pour se déconnecter de linux1 est exit,x
    • La commande pour copier un fichier est cp,x
    • La commande pour renommer un fichier est mv,x
    • La commande pour déplacer un fichier est mv,x
    • La commande pour changer de dossier courant est cd,x
    • La commande pour créer un répertoire est mkdir,x
    • La commande pour visualiser le contenu d'un fichier sans l'éditer est cat,x
    • La commande pour voir quel est le dossier courant est pwd,x
    • La commande pour détruire un fichier est rm,x
    • La commande pour détruire un dossier vide est rmdir,x

    La ligne de commande

    • Qu'est-ce qui permet de distinguer / séparer les différentes parties d'une commande ? l'espace,x
    • Dans la commande vitest, combien y a-t-il de parties ? 1,x

    Les commandes vi

    Quelle commande vi permet faire l'action suivante ?
    • de passer du mode commande au mode insertion i,x
    • de passer du mode commande au mode exécution :,x
    • de revenir au mode commande esc,x
    En mode commande :
    • d'effacer un caractère x,x
    • de couper la ligne sous le curseur dd,x
    • de copier la ligne sous le curseur yy,x
    • de coller la ligne précédemment copiée ou collée p,x
    En mode exécution (n'oubliez pas de commencer par la commande qui permet de passer au mode exécution) :
    • de sauver sans quitter l'éditeur :,x w,x
    • de sauver le fichier sous le nom "exercice" :,x w,x exercice,x
    • de quitter sans sauver :,x q!,x
    • de quitter en sauvant :,x x,x

    L'éditeur

    Donnez les commandes nécessaires pour :
    • mettre le mot 'Bonjour' dans un fichier texte appelé MonPremierTD;
    • copier cette première ligne et la coller 3 fois en dessous;
    • sauver et fermer le fichier.
